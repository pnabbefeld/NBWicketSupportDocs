About Wicket Support

Wicket support module for NetBeans IDE. Create a web application and then, in the Frameworks panel, specify packages and files that will make up your Wicket application. If no Wicket library is registered in the IDE, you can register it in the Frameworks panel. During development, when Wicket IDs don't match, you get an error annotation. You can hyperlink from the HTML side to the Java side. There are also file templates and Wicket samples integrated via this module.
